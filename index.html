About the Event ★

SMCC-CCIS Week 2025 is a major campus celebration designed for students, faculty, and guests. It features academic showcases, coding contests, e-sports, exhibits, and fun-filled activities. This event promotes collaboration, creativity, and leadership & gives students a chance to shine.

Event Highlights

  • Programming contest → challenge your coding skills
  • UI/UX exhibit with student projects
  • Quiz bowl, workshop sessions, and awarding ceremony
  • Open registration for all CCIS students & invited guests

Registration URL Example

Example URL:
https://myschool.edu/events/register?event=SMCC%2DCCIS%20Week%202025&section=A

In URL encoding, %20 means a space. Encoded values help make web addresses valid and readable by browsers.

HTML vs XHTML Example

HTML style:

<br>
<img src="images/logo.png" alt="Event Logo">

XHTML style:

<br />
<img src="images/logo.png" alt="Event Logo" />

In XHTML, tags must be properly closed and attributes must be quoted.
